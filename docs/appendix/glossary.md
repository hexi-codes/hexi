# Glossary

## Adapter
A concrete implementation of a port for side effects.

## Port
A protocol/interface defined in `hexi.core.ports`.

## ActionPlan
Strict JSON plan from model describing one-step actions.

## Event
Structured runtime record with `type`, `summary`, `blocking`, and `payload`.

## Runlog
Append-only JSONL file of emitted events for a run.

## Policy
Config block that controls command execution and context limits.
